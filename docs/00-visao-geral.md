# Visão Geral do Projeto

## 1. Nome do Projeto

**WTR Energia Solar - Site Institucional**

## 2. Objetivo

Desenvolver um site institucional moderno, responsivo e otimizado que transmita confiança, inovação, tecnologia, sustentabilidade, credibilidade e profissionalismo, com foco principal em gerar novos clientes e facilitar a solicitação de orçamentos.

## 3. Escopo

### 3.1 Páginas Principais

- **Home** - Página inicial com todas as seções
- **Serviços** - Detalhamento dos serviços oferecidos
- **Sobre** - Página institucional
- **Projetos** - Galeria de projetos realizados
- **Contato** - Formulário de contato

### 3.2 Funcionalidades Principais

- Hero Section com animações e CTA
- Assistente Comercial IA (preparado para integração)
- Formulário de captação de leads
- Galeria de projetos
- Seção de depoimentos
- FAQ interativo
- Responsividade completa

## 4. Público-Alvo

- Proprietários de residências
- Empresas e indústrias
- Pequenos e médios negócios
- Pessoas interessadas em sustentabilidade
- Regiões atendidas pela WTR Energia Solar

## 5. Diferenciais do Projeto

- Design System completo com Design Tokens
- Arquitetura preparada para IA
- Performance otimizada
- SEO avançado
- Acessibilidade WCAG
- Componentização reutilizável

## 6. Métricas de Sucesso

- Taxa de conversão de visitas em orçamentos
- Tempo de carregamento < 3s
- Score Lighthouse > 90
- Acessibilidade > 95%
- Mobile-first implementado

## 7. Restrições

- Não utilizar textos, imagens ou identidade visual de referências externas
- Toda a identidade visual será criada do zero
- Preparar arquitetura para futura integração com OpenAI
- Seguir padrões profissionais de desenvolvimento

## 8. Análise da Referência (refs/index.html)

### 8.1 Estrutura Visual Identificada

**Header:**
- Logo à esquerda
- Informações de contato (e-mail, endereço, telefone)
- Menu de navegação com dropdown
- Botão CTA "FAÇA UM ORÇAMENTO"

**Hero Section:**
- Título de impacto: "Economia e sustentabilidade"
- Subtítulo explicativo
- Vídeo/imagem de fundo
- Animações fadeInUp

**Seções Principais:**
- Benefícios com ícones (Energia Limpa, Fonte silenciosa, Elevada vida útil, Retorno do investimento, Valorização do imóvel, Economia Imediata)
- Comparação Antes/Depois
- Financiamento com carrossel de parceiros
- Projetos com abas (Residencial, Comercial, Industrial)
- Depoimentos com carrossel
- Parceiros (WEG)

**Footer:**
- Logo
- Informações de contato
- Links rápidos
- Newsletter
- Copyright

### 8.2 Tecnologias Utilizadas na Referência

- WordPress
- Elementor
- Font Awesome (ícones)
- Swiper.js (carrosséis)
- Google Tag Manager
- YouTube Embed

### 8.3 Inspirações para Implementação

- Layout em grid com colunas flexíveis
- Animações de entrada (fade in up)
- Carrosséis para depoimentos e parceiros
- Abas para segmentos (Residencial, Comercial, Industrial)
- Seção de benefícios com ícones
- Comparação visual Antes/Depois
- Formulário de newsletter no footer
