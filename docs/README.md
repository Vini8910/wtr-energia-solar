# WTR Energia Solar - Documentação do Projeto

## Visão Geral

Este repositório contém a documentação completa do site institucional da **WTR Energia Solar**, empresa especializada em soluções de energia solar.

## Estrutura da Documentação

| Arquivo | Descrição |
|---------|-----------|
| `00-visao-geral.md` | Visão geral do projeto, objetivos e escopo |
| `01-requisitos.md` | Requisitos funcionais e não funcionais |
| `02-arquitetura.md` | Arquitetura técnica e decisões de design |
| `03-design-system.md` | Design System e componentes reutilizáveis |
| `04-branding.md` | Identidade visual e diretrizes de marca |
| `05-estrutura-front-end.md` | Estrutura e organização do front-end |
| `06-componentes.md` | Biblioteca de componentes |
| `07-chat-ia.md` | Arquitetura do Assistente Comercial IA |
| `08-seo-performance.md` | Estratégias de SEO e performance |
| `09-roadmap.md` | Roadmap de desenvolvimento |
| `10-sprints.md` | Planejamento das sprints |
| `11-checklist.md` | Checklist de implementação |
| `12-definition-of-done.md` | Definition of Done (DoD) |

## Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização com variáveis CSS (Design Tokens)
- **JavaScript (ES6+)** - Interatividade e lógica de negócios
- **PHP 8+** - Backend e processamento de formulários
